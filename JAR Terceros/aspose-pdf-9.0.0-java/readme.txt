
Aspose.Pdf for Java, Generate and manipulate PDF documents
==========================================================================

This distribution contains the following directories:

- lib      : the jar, compiled and ready to use. 
           : aspose-pdf.jar  is autoported release version for JDK 1.6 or higher
	   : aspose-pdf-jdk14.jar is product release specific for JDK 1.4 and 1.5


- javadoc  : the documentation in html format related to legacy Aspose.Pdf for Java.
- examples : Aspose Examples Dashboard
- xml	   : an XSD file showing Aspose.Pdf product structure
- license  : EULA for product


To get started using Aspose.Pdf for Java, please visit = http://www.aspose.com/docs/display/pdfjava/Getting+Started
